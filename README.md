# TRENDS — Hottest Fashion Premier

A modern, visually rich static fashion blog showcasing trending articles, runway highlights, seasonal guides, and more. Built with semantic HTML and CSS, the site features animated background elements, responsive cards for articles, and an eye-catching header designed for fashion-focused content.


## Features

- Clean, responsive layout with a grid of article cards
- Animated background accents and header shine effect
- Hover interactions and smooth card entrance animations
- Dedicated sections for categories like Streetwear, Sustainable Fashion, Runway, Accessories, and more
- Minimal dependencies — pure HTML + CSS (images hosted locally or via CDN)
- Mobile-friendly with breakpoints for tablets and phones
